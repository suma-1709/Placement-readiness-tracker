# Placement Readiness Tracker

A full-stack web application designed to help students prepare for placements efficiently by tracking DSA progress, soft skills, resume performance, and interview readiness in one place.

## Features

* Track Data Structures & Algorithms (DSA) progress
* Manage soft skills preparation tasks
* Resume ATS score checking system
* Student progress dashboard
* Upload and manage preparation resources
* SQLite database integration for storing progress data
* Responsive frontend using HTML, CSS, and JavaScript

## Tech Stack

### Backend

* Python
* Flask
* SQLite

### Frontend

* HTML
* CSS
* JavaScript

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/harshav0481/placement-readiness-tracker.git
```

### 2. Navigate to the project folder

```bash
cd placement-readiness-tracker
```

### 3. Create a virtual environment

```bash
python -m venv venv
```

### 4. Activate the virtual environment

#### Windows

```bash
venv\Scripts\activate
```

#### Mac/Linux

```bash
source venv/bin/activate
```

### 5. Install dependencies

```bash
pip install -r requirements.txt
```

### 6. Run the application

```bash
python app.py
```
